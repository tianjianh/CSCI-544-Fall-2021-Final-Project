# Machine Translation
 
Implementation for NLITask

## Requirements
- Python >= 3.5
- PyTorch >= 1.3
- nltk >= 3.5
- numpy >= 1.15

## Getting Started
```
python train.py --ffd conv1d --poe True
python train.py --ffd conv1d --poe False
python train.py --ffd linear --poe True
python train.py --ffd linear --poe False
```
